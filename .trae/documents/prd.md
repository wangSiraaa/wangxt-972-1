## 1. 产品概述

健身房私教课包排课与扣课账务系统，围绕课包余额、有效期、教练排班、扣课流水和异常补偿五大核心能力，为会籍顾问、教练、会员和财务人员提供全流程数字化管理。系统确保余额不可直接修改、必须由流水重放计算，并支持家庭共享、企业团课、跨门店预约、月末关账等高级场景。

- 目标用户：健身房会籍顾问、私教教练、会员、财务人员
- 核心价值：消除手工记账差错，实现课包余额可审计、可追溯、可对账

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 会籍顾问 | 系统分配账号 | 维护会员课包、赠课、冻结期、有效期、转课/退课规则 |
| 教练 | 系统分配账号 | 维护授课时段、休假、临时停课、查看可预约容量 |
| 会员 | 手机号登录 | 预约课程、查看课包余额与到期、家庭共享管理 |
| 财务人员 | 系统分配账号 | 月末关账、财务对账、流水重放校验、审计导出 |

### 2.2 功能模块

1. **总览仪表盘**：全系统关键指标概览、待办事项、今日预约/排课/异常汇总
2. **顾问端 - 会员管理**：会员档案、课包维护（多课包/赠课/冻结期/有效期/转课/退课）、家庭共享课包绑定、企业团课额度分配
3. **顾问端 - 课包管理**：课包创建/续期/赠送/冻结/解冻/转让审批/退课退款
4. **教练端 - 排班管理**：授课时段设定、休假申请、临时停课、可预约容量查看、批量释放课时
5. **教练端 - 课表视图**：日/周课表、课程状态、学员信息、课时统计、佣金查看
6. **会员端 - 预约中心**：课程预约、候补排队、预约确认（展示扣课来源）、取消预约、家庭共享预约
7. **会员端 - 我的课包**：课包余额、到期日、冻结期、余额推演、扣课来源追溯
8. **财务端 - 流水账务**：正向流水/冲正流水/补偿流水/关账快照、课包流水明细、余额重放校验
9. **财务端 - 月末关账**：关账执行、关账快照生成、关账后调整单、差异清单
10. **扣课模拟器**：模拟跨课包扣减、预演余额变化、展示操作前后差异
11. **系统管理**：课程等级、场地容量、跨门店配置、操作审计、并发预约冲突提示

## 3. 核心流程

### 3.1 会员预约扣课流程

会员选择课程 → 系统检查时间冲突/教练容量 → 匹配课包（按到期顺序）→ 展示扣课来源和余额推演 → 会员确认 → 生成正向流水 → 更新课包余额 → 预约成功

```mermaid
flowchart TD
    "会员选择课程" --> "检查时间冲突"
    "检查时间冲突" --> "检查教练容量"
    "检查教练容量" --> "匹配课包(按到期顺序)"
    "匹配课包(按到期顺序)" --> "展示扣课来源和余额推演"
    "展示扣课来源和余额推演" --> "会员确认预约"
    "会员确认预约" --> "生成正向扣课流水"
    "生成正向扣课流水" --> "重放计算余额"
    "重放计算余额" --> "预约成功"
    "检查时间冲突" -->|"冲突"| "返回冲突提示"
    "检查教练容量" -->|"容量已满"| "加入候补队列"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深墨绿 (#0D2818) + 金色点缀 (#D4A843)，传递专业与品质感
- 辅助色：暖灰 (#F5F3EF) 背景、深灰 (#1A1A2E) 卡片、珊瑚红 (#E85D4A) 警示
- 按钮风格：圆角 (8px)，主操作实心金色，次操作描边，危险操作红色
- 字体：标题使用 Playfair Display，正文使用 DM Sans
- 布局风格：左侧导航 + 右侧内容区，卡片式模块布局
- 图标：Lucide 图标库，线条风格
- 动效：页面切换渐入 (200ms)、卡片悬停微升 (translate-y -2px)、数据刷新脉冲

### 4.3 响应式设计

- 桌面优先设计，最小宽度 1280px
- 平板适配：侧边栏可折叠，卡片改为2列
- 移动端适配：底部导航，卡片单列，关键操作浮窗

### 4.4 动效设计

- 页面加载：卡片依次渐入 (stagger 100ms)
- 数据变更：数字跳动动画 (countUp)
- 关账操作：进度条+脉冲光效
- 余额推演：差异项闪金高亮
- 冲突提示：抖动+红色边框
